# Аналитика и синтез ответов

UI cache: `?v=95` (при странном поведении — **Ctrl+F5**).

Полная версия: `Docs/25_analytics_synthesis.md`.

## Шаблон ответа

```markdown
## Сводка
## Источники по группам (метод / год / география)
## Консенсус и разногласия
## Уверенность (N источников, уровень достоверности)
## Пробелы в знаниях
## Рекомендации (кейсы, эксперты, темы)
## Уточняющий вопрос
```

## UI

- Collapsible `##`-разделы; «Сводка» открыта.
- **Пояснить** — у каждого раздела и под ответом.
- **Обновить** — повтор на исходный вопрос (`regenerate`).
- **Источники** — свёрнуты по умолчанию.
