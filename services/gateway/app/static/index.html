Как обработать файл?

      

      

        
        
        
      

      
Полный путь: OCR → MD → граф L1–L6 → Neo4j → Qdrant → L4 HDBSCAN. Лёгкий: OCR → MD → Qdrant для поиска в чате (без графа и L4).
