Загружайте документы, стройте граф Neo4j и Qdrant, задавайте вопросы AI-агентам по ролям. Разделы доступны в верхнем меню.

              Что такое аномалии?
              
    
                
    Аномалии — L4-факты из mkg_claims, которые HDBSCAN не включил ни в плотный кластер (cluster_id = -1, is_anomaly = true). На карте — красные точки; клик открывает панель справа: текст слева, связи Neo4j справа.
    
                
    Кластеры — цветные области и карточки ниже. Клик по области или карточке — описание темы и связи между фактами кластера.
    
                
    Если все точки красные (0 кластеров): мало L4-данных или эмбеддинги слишком разные. Уменьшите HDBSCAN_MIN_CLUSTER_SIZE в .env, перезапустите gateway и нажмите «Перекластеризовать».
    
                
    Подробнее: L4-аномалии и anomaly_mode · пайплайн L4

    Документация MKG
    
            
    Архитектура пайплайна, агентов и ролей — в приложении, без внешних ссылок.
    
          
    
          
    
          
    
            
    
              
    
              
    
              
    
              
    
            
    
          
    
        
    
      
    

      
    
        
    
          
    
            
    Настройки
    
            
    Модели пайплайна, обзор знаний и подписки на темы (MVP).
    
          
    

          
    
            
    Оформление
    
            
    
              
    
                Язык интерфейса
                
    
                  
                  
                
    
              
    
              
    
                Тема
                
    
                  
                  
                
    
              
    
            
    
          
    

          
    
            
    Экспорт данных
    
            
    Скачайте обзор, риски, документы, граф, аномалии L4 и диалоги — все форматы экспорта в одном месте.
    
            
    
          
    

          
    
            
    Подписки на темы (MVP)
    
            
    Ключевые слова через запятую. Toast при загрузке документа с совпадением в названии. Roadmap: email/push.
    
            
            
            
            
          
    

          
    
            
    Доступ к данным
    
            
    Какие роли видят документы каждого грифа. Администратор всегда имеет полный доступ.
    
            
    
            
    
              
              
            
    
          
    

          
    Модели
    
          
    Модель для каждого сервиса пайплайна. Сохраняется в Postgres и применяется к новым задачам.
    
          
    
            
    
              
    Извлечение графа
    
              
    LLM для извлечения L1–L6 и фильтрации чанков при ingestion.
    
              
              
            
    
            
    
              
    OCR (ingestion)
    
              
    Распознавание PDF и изображений. «auto» — выбор по типу файла.
    
              
              
            
    
            
    
              
    Qdrant — индексация
    
              
    Embedding doc при записи точек в коллекции.
    
              
              
            
    
            
    
              
    Qdrant — поиск
    
              
    Embedding query при семантическом поиске.

    Как обработать файл?
    
      
    
      
    
        
        
        
      
    
      
    Файл будет загружен и отправлен вместе с текстом при нажатии «Отправить». Полный путь: OCR → MD → граф → Neo4j → Qdrant. Лёгкий: OCR → MD → Qdrant (без графа).
